安装git

安装nodejs npm
Press here to download node
解压

建立软链接

cnpm(国内淘宝镜像)

安装hexo

创建目录(任意位置)

初始化博客

本地预览(浏览器输入localhost:4000)

部署github

首先，你先要有一个GitHub账户，去注册一个吧。
注册完登录后，在GitHub.com中看到一个New repository，新建仓库
创建一个和你用户名相同的仓库，后面加xxxx.github.io，只有这样，将来要部署到GitHub page的时候，才会被识别，也就是xxxx.github.io，其中xxxx就是你注册GitHub的用户名。

name写上面的xxxx
email写xxxx.github.io

打开站点配置文件 _config.yml(就是刚才那个blog文件夹下的)

安装deploy-git(这样你才能用命令部署到GitHub)

hexo clean清除了你之前生成的东西，也可以不加。
hexo generate 顾名思义，生成静态文章，可以用 hexo g缩写
hexo deploy 部署文章，可以用hexo d缩写
